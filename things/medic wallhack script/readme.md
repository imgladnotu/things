<h2>what is it?</h2>

a script that lets you see your teammates through walls as a medic by abusing `hud_medicautocallersthreshold`<br>
<h2>is it any good?</h2>

pros to this mod:<br>

* you can see all of your teammates through walls

cons to this mod:<br>

* limited to medic
* limited by time

<h2>what is it?</h2>

makes shadows beneath players super duper massive using a hud trick.<br>
<h2>is it any good?</h2>

pros to this mod:<br>

* works on `sv_pure 2`
* VERY big shadows
* works on all operating systems
* undetectable

cons to this mod:<br>

* requires game restart to change
* requires hud editing
* requires a hud
